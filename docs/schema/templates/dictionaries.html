4.6 Dictionaries

The EN ISO 23387 defines a standardized way to express construction data with the concept of data templates. Such data templates can be used to structure the information, based on external standards. The data templates in EN ISO 23387 make use of the data structure, defined in ISO 12006-3. Data templates include well defined properties according to EN ISO 23386. The properties and the data template can be stored in central dictionaries, from where they can be retrieved and used in projects.

To use the data templates from (Online-)Dictionaries in the construction objects data view, the data templates have to be mapped from the structure ISO 12006-3 to the structure of EN ISO 16739.1:2018.
