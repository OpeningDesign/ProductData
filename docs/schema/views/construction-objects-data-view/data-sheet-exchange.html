1.1.2 Data sheet exchange

The "Data sheet exchange" describes how data of specific completely defined constructions objects (e.g. building products) can be structured and exchanged. The description of data sheets for construction objects requires that the data templates as defined above have been agreed upon and are applied.

Data Sheets can be extracted from Catalogue of configured construction objects based on the selection.

With data sheets, specific construction objects can be described digitally, e.g. a digital description of concrete, existing, completely defined products (e.g. heating types) or real existing product instances (e.g. heating terminals), based on the agreed data templates. Construction Object Data Sheets contain:

  • Descriptive characteristics based on the characteristic definition taken from the Data Templates
  • Descriptive attached documents and their metadata, such as PDF files, photos and other files with further information
  • References to one or more classification systems (such as Omniclass or Uniclass, etc.)
